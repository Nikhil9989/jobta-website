# Images Directory

This directory should contain all the image assets for the JOBTA website.

## Required Images:

1. `jobta-logo.png` - The main logo file (should be in black against transparent background)
2. `hero-bg.jpg` - Background image for the hero section
3. `about-img.jpg` - Image for the About Us section
4. `employers-img.jpg` - Image for the Employers section
5. `job-seekers-img.jpg` - Image for the Job Seekers section

## Image Guidelines:

- All images should be optimized for web (use formats like .jpg, .png, or .webp)
- Recommended hero image size: 1920x1080px (16:9 ratio)
- Recommended section images: 800x600px or 600x400px
- Logo should be provided in PNG format with transparent background
- Consider providing multiple sizes of the logo for different screen resolutions
